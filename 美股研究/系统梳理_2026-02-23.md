## 🛡️ 风控系统

### 默认风控参数
```python
止损线：-20%           # 跌破触发CRITICAL警报
止盈提醒：+50%         # 达到触发WARNING提醒
最大单仓位：25%        # 超过触发仓位警告
```

### 风险级别
- **🚨 CRITICAL**: 止损触发 → 建议立即处理
- **⚠️ WARNING**: 止盈提醒 / 仓位过重 → 考虑调整
- **✅ OK**: 正常范围内

### 警报输出
```bash
/tmp/stock_risk_alert.json
格式: 
{
  "time": "2026-02-23T10:00:00",
  "alerts": [
    {
      "level": "CRITICAL",
      "ticker": "NVDA",
      "type": "STOP_LOSS",
      "message": "🚨 NVDA 触发止损！当前亏损 -22%",
      "current_price": 780.5,
      "pnl_pct": -22.0
    }
  ]
}
```

## 📈 分析工具使用

### 1. 风控检查
```bash
cd ~/.openclaw/workspace/美股研究
python3 risk_manager.py check        # 检查所有持仓
python3 risk_manager.py summary      # 持仓总览
python3 risk_manager.py add          # 添加持仓（交互式）
```

### 2. 个股分析
```bash
python3 stock_analyst.py NVDA        # 快速分析NVDA
python3 stock_analyst.py CEG --full  # 完整分析（含新闻）
python3 stock_analyst.py --all       # 分析全部观察名单
```

### 3. 市场情报
```bash
python3 market_intel.py              # 生成每日情报简报
```
