# 核能投资逻辑 ☢️

> AI驱动数据中心电力需求 → 核能是唯一可行的稳定零碳电源

---

## 核心论点

```
AI算力爆炸 
    → 数据中心用电量激增（+160% by 2030）
    → 可再生能源间歇性无法满足24/7算力需求
    → 核能成为唯一可行的稳定零碳电源
    → 核能资产重新定价
```

### 关键数据

- Goldman Sachs：数据中心电力需求2030年前增长 **160%**
- 单个大型数据中心用电量 = 一个小型城市
- 可再生能源间歇性无法满足AI算力24/7需求

## 风险清单

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 核事故 | 低 | 极高 | 分散持仓，不重仓单一核电标的 |
| 监管政策收紧 | 中 | 高 | 关注政策动向 |
| SMR商业化延迟 | 高 | 中 | 避免重仓SMR初期公司 |
| 替代能源技术突破 | 低 | 中 | 定期评估 |
| 天然气价格暴跌 | 中 | 中 | 关注能源市场 |
