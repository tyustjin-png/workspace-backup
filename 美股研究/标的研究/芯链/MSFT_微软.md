## 投资逻辑

### 看多理由（Bull Case）
1. **Azure AI云高速增长**：AI工作负载推动云增速重新加速，50%+增速
2. **OpenAI独家合作**：最先进AI模型的商业分发权
3. **Copilot变现**：Office 365 Copilot每用户+$30/月，渗透率持续提升
4. **企业IT预算黏性**：Windows + Azure + M365深度绑定企业，迁移成本极高
5. **核能布局**：与Constellation签20年协议，确保数据中心能源安全

### 风险（Bear Case）
1. **估值不便宜**：相对增速PE偏高
2. **AI竞争加剧**：Google Gemini、Amazon AWS AI追赶
3. **OpenAI关系不确定**：OpenAI独立性增强后合作条款可能变化
4. **监管风险**：欧盟反垄断压力
