# 机会评估框架（反脆弱原则）

> 对每个新机会，用这个框架评估再决策。

---

## 快速评分表（复制使用）

### 1. 不对称性（下行有限，上行巨大？）

| 评估项 | 问题 | 评分(1-5) |
|--------|------|-----------|
| 最大损失 | 如果完全失败，损失多少？ | |
| 最大收益 | 如果完全成功，收益多少？ | |
| 收益/风险比 | 比值 > 10倍？ | |

**决策**：
- 收益/风险比 > 10：值得尝试
- 收益/风险比 < 3：谨慎
- 收益/风险比 < 1：放弃

---

### 2. 能力匹配（与现有技能相关？）

| 评估项 | 问题 | 评分(1-5) |
|--------|------|-----------|
| 技能复用 | 现有技能可复用？ | |
| 学习曲线 | 学习成本高吗？ | |
| 竞争优势 | 你比别人强在哪？ | |

**决策**：
- 高度匹配（4-5分）：优先
- 中度匹配（3分）：值得学习
- 低度匹配（1-2分）：除非有超高回报

---

### 3. 参与门槛（能以小成本买票入场？）

| 评估项 | 问题 | 评分(1-5) |
|--------|------|-----------|
| 资金门槛 | 需要多少资金？ | |
| 时间门槛 | 需要投入多少时间？ | |
| 最小可行测试 | 能小规模测试吗？ | |

**决策**：
- 低门槛（可小规模测试）：立即尝试
- 中门槛：需要更多研究
- 高门槛：需要非常高的确信度

---

### 4. 退出灵活（可以随时退出？）

| 评估项 | 问题 | 评分(1-5) |
|--------|------|-----------|
| 流动性 | 资产可以快速变现？ | |
| 沉没成本 | 投入的时间/资金能回收？ | |
| 心理成本 | 放弃容易吗？ | |

**决策**：
- 高灵活性：更愿意尝试
- 低灵活性：需要更高确信度

---
