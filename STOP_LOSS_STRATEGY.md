# 🛡️ 止盈止损策略 v2.1

**更新时间：** 2026-02-01 00:23 GMT+8

---

## 🎯 策略核心：翻倍回本，剩余持有

**理念：**
- ✅ 翻倍时卖出 50% 回本（本金安全）
- ✅ 剩余 50% 继续持有（追求更高收益）
- ✅ 已回本部分：只有止损，无止盈

---

## 📊 完整流程

### 阶段 1：买入后（持有 100%）

**监控条件：**
- 🔴 **-50% 止损** → 全部卖出
- 🟢 **+100% 翻倍** → 卖出 50%（进入阶段 2）
- ⏰ **超时止损** → 持有 >48h 且盈利 <50%，全部卖出

**示例：**
```
买入：1.0 SOL @ $0.0001
当前价：$0.00005（-50%）
触发：止损，卖出 100%
结果：亏损 0.5 SOL
```

```
买入：1.0 SOL @ $0.0001
当前价：$0.0002（+100%）
触发：翻倍回本，卖出 50%
结果：回本 1.0 SOL，剩余 50% 继续持有
```

---

### 阶段 2：已回本（持有 50%）

**监控条件：**
- 🔴 **-50% 止损** → 剩余 50% 全部卖出
- ❌ **无止盈** → 继续持有，追求更高收益
- ❌ **无超时止损** → 永久持有（只要不跌破止损）

**示例：**
```
已回本状态：
  原始买入：1.0 SOL @ $0.0001
  已卖出：50% @ $0.0002（回本 1.0 SOL）
  剩余持有：50%，成本为 0（已回本）

场景 A：继续上涨
  当前价：$0.0005（+400%）
  操作：继续持有 ✅
  收益：1.0 SOL（回本）+ 剩余价值

场景 B：回调止损
  当前价：$0.00005（-50% from buy price）
  操作：卖出剩余 50% 🔴
  收益：1.0 SOL（回本）+ 少量剩余
```

## ⚠️ 风险管理

### 止损保护（全程有效）

**-50% 止损始终有效：**
- 未回本：止损保护本金
- 已回本：止损保护利润

**不会发生的情况：**
- ❌ 无限期亏损
- ❌ 回本后再次深亏

### 超时止损（仅未回本时）

**触发条件：**
- 持有 > 48 小时
- 盈利 < 50%
- 未回本（partial_sold = False）

**已回本后：**
- ✅ 不触发超时止损
- ✅ 可以长期持有
- ✅ 追求长期暴涨

## 🔧 技术实现

### 持仓状态字段

```json
{
  "contract": "...",
  "token_name": "$XXX",
  "amount_sol": 1.0,
  "buy_price": 0.0001,
  "partial_sold": false,  // false = 未回本，true = 已回本
  "status": "holding"
}
```

### 检查逻辑

```python
# 未回本阶段
if not partial_sold:
    if profit_pct <= -0.50:
        sell(100%)  # 止损
    elif profit_pct >= 1.00:
        sell(50%)   # 翻倍回本
        partial_sold = True
    elif hold_hours > 48 and profit_pct < 0.50:
        sell(100%)  # 超时止损

# 已回本阶段
else:  # partial_sold = True
    if profit_pct <= -0.50:
        sell(remaining 50%)  # 只有止损，无止盈
```
