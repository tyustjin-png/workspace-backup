- ✅ 持仓实时监控
- ✅ 自动卖出执行（待完成 Jupiter Swap）

**策略演变：**

**初始策略：**
```
+50% → 卖出 50%
+100% → 卖出剩余 50%
结果：全部清仓
```

**最终策略（应你要求）：**
```
+100% → 卖出 50% 回本
剩余 50% → 继续持有（无止盈）
只有 -50% 止损保护
```

**策略优势：**
- ✅ 翻倍回本保证本金安全
- ✅ 剩余持有追求 10x-100x 暴涨
- ✅ 适合 Meme 币的暴涨特性
- ✅ "免费筹码"长期持有

   - 硬性门槛会错失机会
   - 梯度评分更合理
   - 早期币需要特殊规则

   - 过早止盈错失暴涨

   - Meme 币适合长期持有（已回本后）

3. **自动化要可靠**
   - 限流保护必不可少
   - 状态持久化避免重复
   - 错误处理要完善

### 策略经验

1. **早期币识别**
   - 上线 < 3 小时

   - AI 代理背书

2. **持仓管理**
   - 翻倍回本保证本金
