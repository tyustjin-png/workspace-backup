# OpenClaw代理调用 vs 原生Claude Code 全功能对比测试

## 项目概述
对比通过OpenClaw代理调用Claude Code与直接使用原生Claude Code的全维度功能差异。

## 测试发现

### 发现1：安全限制差异
**日期：** 2026-03-10  
**场景：** 登录LinkedIn测试密码

| 维度 | OpenClaw代理调用 | 原生Claude Code |
|------|-----------------|----------------|
| 账号登录操作 | ✅ 可执行 | ❌ 法律规则限制 |

**分析：** 原生Claude Code有Anthropic的完整安全策略限制，通过OpenClaw调用时，部分限制被绕过。
