# 书籍下载与深度分析框架

**创建日期：** 2026-03-10  
**触发方式：** 金哥给出书名或人物名

---

## 📥 第一步：下载书籍

### 搜索来源（优先级排序）
1. Z-Library / Anna's Archive（最全）
2. LibGen（学术/技术书多）
3. 中文站点：hssgweb.com、yabook.org 等
4. GitHub raw files（有时有资源）

### 下载流程
1. `web_search` 搜索 "书名 + epub/pdf + 下载"
2. `web_fetch` 获取下载页面
3. `exec` + curl/wget 下载文件
4. 存储到 `books/<主题>/` 目录

### 支持格式
- **EPUB**（最佳）→ unzip 解压提取 xhtml
- **PDF**（次选）→ pdftotext 提取文本
- **扫描版PDF** → 识别率一般，文字版最佳

---

## 📊 第二步：深度分析

### 分析框架（四维度）
1. **独特特质** — 这个人/主题的核心差异化
2. **黑暗低谷** — 如何度过困难时期
3. **成功方法** — 可复用的方法论
4. **持续成长** — 保持进化的机制

### 输出格式
每本书单独分析 → `analysis/0X_书名.md`
- 核心发现
- 关键案例
- 金句提取
- 独特洞察

### 最终整合
综合报告 → `analysis/99_综合洞察.md`
- 跨书籍交叉分析
- 核心精华提炼
- 可借鉴行动清单
- 20条金句

---

## 📤 第三步：交付

1. 分段发送报告精华（Telegram消息）
2. 发送完整文件供下载（Telegram文件）
3. 服务器存档 `books/<主题>/analysis/`

---

## 🔧 技术备注

- PDF超过100页：用 pdftotext 提取，不用 pdf 工具
- EPUB：`unzip book.epub -d /tmp/epub_dir`
- 长文本分段处理，避免上下文溢出
- 中文书优先找中文版
