## 二、核心思想脉络

### 1. 分区间定投 → 分权重定抛：完整投资方法论的闭环

这是本批次最重要的思想成果。金哥在多篇笔记中反复打磨这套体系（#812, #818, #824），从熊市买入到牛市卖出形成完整闭环：

**分区间定投（买入端）**：
- 根据组合指标对周期完全划分，在最好及相对更好的位置投入更多份额
- 解决的是熊市"心理和执行两个层面"的卡点

**分权重定抛（卖出端）**：
框架为四个维度：
> **维度**——什么/谁比较重要
> **权重**——什么/谁有多重要
> **概率**——什么/谁提示了信号吗？确定性有多大
> **仓位**——以上的结论对应多少仓位要卖
— Note #824

**核心公式示例**：选择MA120、BTC达10万刀、美国降息结束三个维度，各权重1/3，跌破MA120但未达10万且降息中 → 卖出1/3仓位。

**量化思维是底层支撑**：
> "量化是从道理到执行的桥梁" — Note #812
> "量化不是精确计算，而是用数据和统计减少不确定性" — Note #818

### 2. 行动优先于认知

这是贯穿多篇写作笔记的核心信念（#770, #771, #772, #773）：

> "学习是个执行动作，却没有很强的结果意义。我更喜欢进化这个词，进化是个含有强结果意义的动作。" — Note #770

> "认知是'通过思想、经验和感官获得知识和理解的心理活动或过程'。其中的过程就是学习、体验和经历，这都是行动。所以，认知在行动中产生。" — Note #772

六条行动原则：完成比完美重要、行动比认知重要、教比学重要、写作比群聊重要、突破源于专注、场外比场内重要。

### 3. 反脆弱与风险管理

> "如果有个可能，会让我们'挂掉'，无论多大的收益，还是应该避免。因为只要一次一次的持续，无论单次概率多低，整体出事的概率都大幅提高。" — Note #835

实际案例：金哥用BTC质押借稳定币参与打新，大跌时险些被清算，深刻反思反脆弱性（#835）。朋友因wbeth质押全账户爆仓的前车之鉴更加强化了这一认知。

### 4. 写作复利论

> "没有任何一个技能像写作一样拥有多重复利。" — Note #771

三重复利：成果积累复利（输出倒逼输入）、成长复利（系列化输出形成思考体系）、影响力复利（熟悉带来好感，好感转化信任）。

### 5. 策略分级思想

引用汪涛观点（#813）：
> "熟悉的策略比完美的策略更重要。很多失败其实是，对于不熟悉的策略，上了过重的仓位。"

C级（成熟策略）> B级（知道但未实践）> A级（刚知道），不同级别对应不同仓位权重。

### 6. 均值回归与马太效应

> "没有富裕之前需要依靠马太效应富裕，富裕之后的首要命题是对抗均值回归。" — Note #813引用汪涛
> "如果放在币圈，总结一句话，就是不停的参与各种机会，把赚到的钱换成更多的大饼。"

## 九、综合分析

**最大的特征是方法论驱动**。无论是买入（分区间定投）、卖出（分权重定抛）、项目参与（策略分级）、还是团队管理（SOP+信用值系统），金哥都试图将模糊的判断转化为可量化、可执行、可迭代的系统。这与他在USER.md中被定义的"量化一切"特征完全吻合。

**最值得注意的进化轨迹**是从纯粹的链上/币圈投资者向"AI+投资+内容"三位一体的转型。2025年的笔记明显增加了AI工具使用（Claude Code、Cursor、RPA工作流）、港美股研究（特斯拉/英伟达/AI产业链）、以及更深层的个人成长（猫叔课程、复利人生大会、盖洛普优势测试）。

**最打动人的一句话**：
> "先完成，再去完美，这是成长效率最高的方式。" — Note #770

这句话也是金哥整个笔记库的精神缩影——不完美但持续行动，在行动中进化。
