## 📊 止损/止盈规则

### 规则 1: 止损 -50%
- **条件**: 亏损达到 50%
- **动作**: **全部卖出**
- **原因**: 防止继续亏损

### 规则 2: 翻倍回本 +100%
- **条件**: 盈利达到 100%（翻倍）
- **动作**: **卖出 50%** 回本，剩余继续持有
- **原因**: 锁定本金，让利润奔跑
- **特点**: 只执行一次，之后只有止损保护

### 规则 3: 超时止损 48h
- **条件**: 持有 48 小时且未翻倍、未回本
- **动作**: **全部卖出**
- **原因**: 避免资金长期占用
- **例外**: 如果已回本（执行过规则 2），不触发超时止损

3. 查询当前价格（DexScreener API）

7. 更新持仓状态
8. 写入通知文件 → Telegram 推送

## 🎯 实际案例

### 场景 1: 止损
```
买入价: $0.001
当前价: $0.0005
盈亏: -50%
持有: 2 小时

→ 触发止损，全部卖出
```

### 场景 2: 翻倍回本
```
买入: 1 SOL
买入价: $0.001
当前价: $0.002
盈亏: +100%
持有: 3 小时

→ 卖出 50%（回本 0.5 SOL）
→ 剩余继续持有（0.5 SOL 的代币）
```

### 场景 3: 回本后继续上涨
```
买入: 1 SOL
买入价: $0.001
翻倍价: $0.002 → 已卖出 50% 回本
当前价: $0.005
盈亏: +400%
持有: 12 小时

→ 继续持有（剩余 50% 代币是纯利润）
→ 只有止损保护（跌破 -50% 才卖）
→ 无止盈上限
```

### 场景 4: 超时止损
```
买入价: $0.001
当前价: $0.0012
盈亏: +20%
持有: 50 小时

→ 超过 48 小时未翻倍
→ 触发超时止损，全部卖出
```

## 💡 策略思路

这个止损系统的核心理念：

1. **控制风险** - 最多亏 50%
2. **锁定本金** - 翻倍后拿回本金
3. **让利润奔跑** - 回本后无止盈上限
4. **避免套牢** - 48 小时超时保护

**适合场景**: Meme 币短线交易，追求早期 10x-100x 机会
