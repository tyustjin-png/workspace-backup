# HEARTBEAT.md - 小定的定时任务

## 每日定投检查（08:00）

1. 读取当前z值
2. 如果 z >= 1：
   - 计算买入倍数（斐波那契）
   - 检查账户余额
   - 执行买入
   - 记录到 `memory/定投日志.md`
   - 通知金哥
3. 如果 z = 0：
   - 记录"今日不买入"
   - 静默（不通知）

## 每周总结（周日 20:00）

1. 统计本周定投数据
2. 分析z值趋势
3. 生成周报发送给金哥

## 异常处理（任何时候）

如果发现：
- API错误
- 余额不足
- 价格异常
- 数据源失效

立即：
1. 停止自动执行
2. 通知金哥
3. 等待人工处理
