# 📱 Apple 备忘录集成指南

## ✅ 格式兼容性

我生成的日程格式**完全兼容** Apple 备忘录的清单功能！

格式说明：
```markdown
- [ ] 未完成任务  → Apple 备忘录显示为 ○（空心圆）
- [x] 已完成任务  → Apple 备忘录显示为 ●（实心圆）
```

## 📋 如何复制到 Apple 备忘录

### 方法1：每日总结时复制（推荐）

晚上 20:00 收到我的每日总结后：
1. 复制整个消息
2. 打开 Apple 备忘录
3. 新建笔记
4. 粘贴
5. ✅ 完成！自动变成可打勾的清单

### 方法2：随时导出

在 Telegram 对我说：
```
导出今日日程到备忘录
```

我会发送格式化好的内容，直接复制粘贴即可。

### 方法3：手动导出（服务器）

```bash
cd ~/.openclaw/workspace
python3 export_to_apple_notes.py

# 指定日期
python3 export_to_apple_notes.py 2026-03-03
```

直接发语音："和某某团队吃了午饭，聊得很开心"

### 2. Emoji 在 Apple 备忘录中完美显示
所有 emoji（⛹️💰✍️🤝🏠）都能正常显示

### 3. 标题层级清晰
```

## ⛹️ 运动                   → 分类标题
- [ ] 100仰卧起坐            → 清单项目
```

- 直接打勾 ✓ 标记完成
- 添加新任务
- 调整顺序
- 然后复制回来告诉我更新（可选）

## ❓ 常见问题

**Q: 复制后格式乱了？**
A: 确保复制完整内容，包括开头的 `#` 标题

**Q: Emoji 显示不正常？**
A: Apple 备忘录完全支持 Unicode Emoji，应该不会有问题

**Q: 可以修改模板吗？**
A: 可以！编辑 `daily_schedule_manager.py` 中的 `SCHEDULE_TEMPLATE`

**Q: 可以添加更多分类吗？**
A: 当然！告诉我需要什么分类，我帮你加上
