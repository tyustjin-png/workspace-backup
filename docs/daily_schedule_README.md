# 每日日程助手 - 使用说明

## 📋 功能概述

自动分类记录每日活动，支持运动、投资、写作、社交、家庭五大类别。

---

## 🎯 使用方式

### 1️⃣ 添加任务（语音或文字）

直接对紫龙说：
- "跑了5公里" → 自动记录到【运动】
- "定投200U" → 自动记录到【投资】  
- "写了封邮件" → 自动记录到【写作】
- "陪儿子玩" → 自动记录到【家庭】
- "和朋友午餐" → 自动记录到【社交/人脉】

### 2️⃣ 标记任务完成

任务添加时会自动标记为完成（✅）

### 3️⃣ 查看日程

**手动查看**：
```bash
cat ~/.openclaw/workspace/memory/daily-schedule/2026-03-05.md
```

**获取汇报**：
```bash
python3 ~/.openclaw/workspace/schedule_report.py
```

---

## 📅 定时提醒（已配置）

每天会在以下时间自动汇报日程：
- **10:00** - 第一次日程汇报
- **15:00** - 第二次日程汇报  
- **19:00** - 第三次日程汇报
- **20:00** - 每日总结

---

## 📊 汇报格式

```
📋 **2026-03-05 日程清单**

## ⛹️ 运动
  • ✅ 100仰卧起坐
  • ✅ 50俯卧撑
  • 🔸 20深蹲

## 💰 投资
  • ✅ 定投相关
  • 🔸 链上数据

## ✍️ 写作
  • 🔸 邮件
  • 🔸 公众号

...

---
**完成度**: 5/15 (33%)
```

**格式说明**：
- ✅ = 已完成
- 🔸 = 待完成（橙色小菱形，柔和）
- 按类别分组
- 最后显示总完成度

---

## 🔧 技术细节

### 文件位置
- **日程文件**: `~/.openclaw/workspace/memory/daily-schedule/YYYY-MM-DD.md`
- **汇报脚本**: `~/.openclaw/workspace/schedule_report.py`
- **管理脚本**: `~/.openclaw/workspace/daily_schedule_manager.py`
- **消息文件**: `/tmp/schedule_message.txt`

### 自动分类规则
系统根据关键词自动分类：
- **运动**: 跑步、健身、俯卧撑、深蹲、游泳、打球...
- **投资**: 定投、BTC、ETH、买入、卖出、链上...
- **写作**: 邮件、公众号、推特、文章、博客...
- **社交/人脉**: 午餐、聚会、见面、会议、交流...
- **家庭**: 罗老师、父母、儿子、孩子、陪...

### 定时任务
```bash
# 每天 10:00, 15:00, 19:00 生成日程汇报
0 10,15,19 * * * /root/.openclaw/workspace/generate_schedule_reminder.sh
```

## 💡 使用技巧

1. **语音输入更方便** - Telegram自动转文字
2. **简短描述即可** - 系统会自动分类
3. **随时记录** - 做完立即记录，不要积压
4. **查看趋势** - 每天看完成度，了解时间分配

---

## 🐛 故障排查

### 问题1：任务分类错误
- 手动编辑日程文件调整
- 或者说得更明确：`【运动】跑步5公里`

### 问题2：未收到定时提醒
- 检查定时任务：`crontab -l | grep schedule`
- 查看日志：`tail /tmp/schedule_reminder.log`

### 问题3：汇报格式不对
- 手动运行测试：`python3 ~/.openclaw/workspace/schedule_report.py`
- 检查日程文件格式是否正确
